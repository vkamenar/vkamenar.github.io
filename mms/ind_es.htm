Compilador de MMS

[ Descargar | Foros ]
El Servicio de Mensajería Multimedia (en inglés, Multimedia Messaging Service) o MMS es una evolución del SMS. Admite texto con imágenes multimedia, secuencias animadas, sonido y vídeo. El contenido visual se muestra y reproduce dentro de la aplicación de mensajería del destinatario.

MMS es compatible con la mayoría de los móviles fabricados desde finales de 2001, incluyendo teléfonos básicos (los de gama baja o características limitadas) y smartphones: Android, Symbian, Blackberry, Windows Phone, entre otros. Apple soporta MMS desde iOS 3, pero hace caso omiso del formateo SMIL, el cual sirve para especificar el diseño visual y transiciones en el contenido del MMS. Algunos dispositivos admiten la combinación de MMS con HTML.

Las especificaciones técnicas de MMS fueron creadas originalmente por el Foro WAP y 3GPP. Actualmente, dichas especificaciones son administradas por la Open Mobile Alliance (OMA). SMIL es el lenguaje utilizado para controlar la manera como se visualizan los MMS. SMIL para MMS es como HTML para las páginas web. La mayoría de las aplicaciones compatibles con MMS admiten un subconjunto de SMIL 2.0. Las versiones más recientes de SMIL no son ampliamente soportadas en el ámbito de MMS.

El Compilador de MMS es una herramienta de código abierto que permite generar archivos del Servicio de Mensajería Multimedia (MMS) a partir de un conjunto predefinido de archivos de entrada: texto, imágenes, audio y vídeo. Esta herramienta es compatible con Windows y Linux (x86). No requiere dependencias ni componentes adicionales.

Por favor, consulte la Introducción y Guía del Usuario para obtener más detalles y recomendaciones prácticas sobre cómo compilar archivos MMS compatibles con distintos dispositivos móviles.

Al publicar un archivo MMS en un servidor web, se deben cumplir las siguientes condiciones:

  • La URL debe ser accesible a través del APN de MMS configurado en el móvil. Si el acceso se encuentra bloqueado por el operador, una solución temporal es configurar el APN de Internet como APN de MMS para que el móvil pueda descargar archivos MMS desde cualquier sitio web público. Esta solución alternativa debe usarse solo para pruebas, ya que descargar y reproducir archivos MMS desde cualquier sitio web podría no ser seguro.
  • El tipo MIME del archivo MMS debe ser application/vnd.wap.mms-message
  • Algunos clientes MMS no admiten redirecciones HTTP. Por lo tanto, no utilice acortadores de URL ni herramientas similares.
  • Los clientes MMS más antiguos no son compatibles con SSL/TLS. Por lo tanto, la URL debe usar HTTP, no HTTPS.

La mayoría de los teléfonos no muestran ningún error si falla la descarga del MMS. Por lo tanto, podría ser necesario usar un depurador (ej., adb para Android) para ver los mensajes de error exactos.
