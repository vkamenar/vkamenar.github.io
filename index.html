Vladimir Kameñar

Electronics Engineer. Professor at the Faculty of Systems Engineering, National University of Colombia, 2004 - 2007. Senior Technical Leader at Thales Group.

 

Publications

  • CriptoNukak: A transport-level communication protocol specialized in the management of medical information, featuring end-to-end encryption, error detection and error correction (2004)
  • The CelerSMS Journal (in Spanish) is a free journal containing technical articles for Android developers (since 2021-05-30)
  • The Unofficial XM File Format Specification covering the original XM (extended module) file format specification and several known sub-formats (2021-08-13)
  • Demoscene, the Digital Art (in Spanish) covers demoscene as an art form, presents its origins, traditions and achievements (2021-09-30)
  • PTTP (Plain Text Telemetry Protocol): An application-level communication protocol for telemetry, using plain text encoding (2022-07-19)

 

Projects

  • uFMOD (or µFMOD): An open-source tiny XM player library written in Assembly language, including XMStrip, a utility tool to reduce the size of an XM file (2006)
  • ImpLib SDK: Combines free programming tools for authoring custom import libraries for Windows DLL, supporting advanced features, like cdecl2stdcall conversion, importing by ordinal, removing original thunk, accessing internal undocumented functions (since 2006)
  • PureBasic OpenAL SDK: A software development kit for PureBasic, featuring the OpenAL audio API, supporting 3D multichannel output of sound sources around the listener (since 2007)
  • Monifa: Bytecode compiler and decompiler for the SIM Application Toolkit, based on the technical standard S@T 01.00 published by SIMalliance (2008)
  • LBS for 2G and 3G: A Location Based System (LBS) for mobile operators, featuring the MLP protocol and computing the location in real time based on the HLR (SS7 protocol) (2010)
  • MMS compiler: A tool to generate Multimedia Messaging Service (MMS) files, including text, images, audio and video (2018)
  • SMS Proxy: Use your mobile as a proxy to send SMS (2019)
  • javamon: Lightweight JVM monitoring agent, compatible with Prometheus (2025)
